# LINE Review Committee 리포트 예제 프로젝트

Line의 Review Committee 코드 품질 개선 기법 시리즈 글을 읽고 예제 및 테스트를 위한 프로젝트

## Line의 Review Committee
```
LY Corporation의 개발 조직에서는 높은 개발 생산성을 유지하기 위해 코드 품질과 개발 문화 개선에 힘쓰고 있습니다. 이와 관련해 여러 활동을 진행하고 있는데요. 그중 하나가 Review Committe입니다(코드 품질과 개발 문화 개선을 위해 저희가 어떤 노력을 하고 있는지 궁금하시다면 DroidKaigi 2021에서 발표한 '오래가는 코드 베이스의 '품질' 문제를 마주하다(일본어)'를 참고하시기 바랍니다). 

Review Committe는 머지된 코드를 다시 한 번 리뷰하면서 리뷰어와 코드 작성자에게 피드백을 제공하는 활동입니다. 이 활동을 통해 얻은 지식과 인사이트를 매주 사내에 Weekly Report라는 이름으로 공유하는데요. Kotlin이나 Swift와 같은 특정 언어에 국한되거나 Android나 iOS와 같은 특정 플랫폼에 국한되는 내용도 있지만, 대부분 언어나 플랫폼에 상관없이 프로그래밍 전반에 적용할 수 있는 주제로 공유하고 있습니다.

이와 같이 Weekly Report 중 일반적으로 널리 적용할 수 있는 리포트를 골라 이 블로그에 '코드 품질 개선 기법'이라는 이름으로 시리즈를 연재하고자 합니다(참고로, 설명하기 위해 작성한 코드는 Kotlin으로 작성한 경우가 많습니다). 아래는 그동안 발행된 '코드 품질 개선 기법' 시리즈의 목록입니다.
```
